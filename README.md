---

## 📋 Sobre o Projeto

O **Granja Admin Sistema** é uma aplicação web que permite gerenciar de forma simples e eficiente:

- ✅ **Cadastro e controle de animais** (bovinos, galinhas, porcos, ovelhas, etc.)
- ✅ **Registro de alimentação** e acompanhamento nutricional
- ✅ **Controle de saúde** (vacinas, tratamentos, consultas veterinárias)
- ✅ **Gestão financeira** (receitas, despesas, fluxo de caixa)
- ✅ **Relatórios e dashboard** para tomada de decisões
