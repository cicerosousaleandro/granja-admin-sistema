#  Sistema de Administração de Granja

![Java](https://img.shields.io/badge/Java-17-blue) ![Spring Boot](https://img.shields.io/badge/Spring_Boot-3.2-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue)

> Sistema completo para gerenciamento de animais e controle financeiro de granjas. Projeto desenvolvido para demonstrar competências em Java Spring Boot, boas práticas de desenvolvimento e arquitetura MVC.

---

## 📋 Sobre o Projeto

O **Granja Admin Sistema** é uma aplicação web que permite gerenciar de forma simples e eficiente:

✅ Cadastro e controle de aves   
✅ Registro de alimentação e acompanhamento nutricional  
✅ Controle de saúde (vacinas, tratamentos, consultas veterinárias)  
✅ Gestão financeira (receitas, despesas, fluxo de caixa)  
✅ Relatórios e dashboard para tomada de decisões
